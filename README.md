## Keepalived

### Install keepalived.
```
apt install keepalived
```
### Start keepalived service.

```
systemctl start keepalived
systemctl enable keepalived
```

### configuration file path.

```
cat /etc/keepalived/keepalived.conf
nano /etc/keepalived/keepalived.conf
```
